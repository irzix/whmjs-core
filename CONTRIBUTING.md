# Contributing to WHMJS

Thanks for taking the time to contribute! 🎉

---

## Getting Started

1. Fork the repository
2. Clone your fork — `git clone https://github.com/irzix/whmjs-core.git`
3. Create a branch — `git checkout -b feature/your-feature`
4. Make your changes
5. Commit — `git commit -m "feat: your feature description"`
6. Push — `git push origin feature/your-feature`
7. Open a Pull Request

---

## What We Need Help With

- 🔌 Payment gateway integrations (Stripe, PayPal, crypto)
- 🖥️ Server module integrations (cPanel, DirectAdmin, Plesk)
- 🌍 Translations and multi-language support
- 🧪 Writing tests
- 📖 Documentation improvements
- 🎨 Frontend (React) components
- 🐛 Bug fixes

---

## Commit Convention

We use [Conventional Commits](https://www.conventionalcommits.org):

```
feat: add stripe payment gateway
fix: resolve invoice calculation bug
chore: update dependencies
docs: improve readme
```

---

## Code Style

- TypeScript only
- Prettier + ESLint — run `pnpm lint` before committing
- Write tests for new features when possible
